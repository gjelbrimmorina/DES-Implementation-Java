# DES Encryption/Decryption Project - Group 7

## Description
This project implements encryption and decryption of text and files using the DES algorithm in Java.

## Features
- Text encryption/decryption
- File encryption/decryption
- Works with images, PDF, etc.

## How to Run
javac *.java
java Main

## Note
DES is insecure today and is used only for educational purposes.
